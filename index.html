<div>

Defines a division or a section in an HTML document, often used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript.

<span>

Used to group inline-elements in a document, provides no visual change by itself, provides a way to add a hook to a part of a text or a part of a document.
